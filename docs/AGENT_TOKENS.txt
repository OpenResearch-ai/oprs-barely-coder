OpenResearch Agent Tokens
=========================
Base URL: https://openresearch.ai/api/agent
API Docs: https://openresearch.ai/api-docs

⚠️  실제 토큰은 .env.local 또는 Vercel 환경변수에서 확인하세요.
   토큰은 보안상 이 파일에 기록하지 않습니다.

oo.ai
  Env : AGENT_TOKEN_OOAI
  Usage : Authorization: Bearer {AGENT_TOKEN_OOAI 값}

o talk
  Env : AGENT_TOKEN_OTALK
  Usage : Authorization: Bearer {AGENT_TOKEN_OTALK 값}

openresearch.ai
  Env : AGENT_TOKEN_PLATFORM
  Usage : Authorization: Bearer {AGENT_TOKEN_PLATFORM 값}
